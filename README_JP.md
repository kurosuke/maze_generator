# mage generator by by python

クラスター法による迷路生成ツール。pythonの練習に作りました。

- cluster algorithm
- 解答は常に1つ
- 入り口と出口は、自分で適当に。
　左上と右下で十分だと思う。

ex) 10x10 の部屋のある迷路を作成

```
% python maze_genrator.py 10 | tee maze.txt
+-+-+-+-+-+-+-+-+-+-+
| |   |       |     |
+ + + + +-+ +-+ +-+-+
|   | |   | |       |
+-+ +-+-+ +-+-+-+ +-+
|   |     |   |     |
+ + + +-+-+ +-+ +-+ +
| |     |       |   |
+-+-+ + +-+-+-+ +-+ +
| | | |   |       | |
+ + + +-+-+ +-+ +-+-+
|       |   |       |
+ + + + + + +-+-+-+ +
| | | | | |   |   | |
+-+ +-+ +-+-+ + +-+-+
|   |     |       | |
+ +-+-+ + + +-+-+ + +
| |     | | | |     |
+-+-+ +-+ +-+ + + + +
|       |       | | |
+-+-+-+-+-+-+-+-+-+-+
```
